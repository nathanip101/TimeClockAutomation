# Time Clock Automation Script
### Skip all the clickinggggg....

#### To use, you must first generate your secrets.yaml file. To initialize your secrets.yaml file, run:
`./clock secret`

#### To clock in, run:
`./clock.sh in`

#### To clock out, run:
`./clock.sh out`

#### To start meal period, run:
`./clock.sh hungry`

#### To end meal period, run:
`./clock.sh full`
